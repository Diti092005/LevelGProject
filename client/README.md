# PermutationGenerator - Client

This is the Angular client application for the Permutation Generator.

## Prerequisites

- Node.js (v18 or higher)
- npm or yarn

## Installation

```bash
npm install
```

## Development Server

```bash
npm start
```

Navigate to `http://localhost:4200/`

## Build

```bash
npm run build
```

Build artifacts will be stored in the `dist/` directory.

## Project Structure

- `src/app/components/` - Angular components
- `src/app/services/` - Services for API and state management
- `src/app/models/` - TypeScript models and interfaces

## Features

- Input screen for permutation configuration
- Display screen for viewing individual permutations
- Pagination view for browsing all permutations
